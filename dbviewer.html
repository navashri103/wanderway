How MongoDB stores data in this project

MongoDB stores data as documents (similar to JSON / Python dictionaries). Related documents are grouped into collections (similar to tables in SQL). This project has 6 collections:

users
Stores registered user accounts
destinations
Trip packages + places to visit
hotels
Hotel listings with pricing
cabs
Cab and bike rental options
combos
All-inclusive travel packages
bookings
Every booking made by users
